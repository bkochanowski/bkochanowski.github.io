About me:

I'm a psychologist with few years of experience in work related to statistical analysis and operating on large data sets. Programming is my hobby. I have knowledge in the field of Python3.x with frameworks: Flask (advanced) and Django (beginner). In my projects I experimented with SQL (MySQL, SQLite3) and NoSQL (MongoDB) databases. I know the basics of the SCRUM, KANBAN and TDD methodology. Currently I am developing a project of a mobile application dedicated for the android platforms using an open source Python library - KIVY. I am looking for professional development opportunities in QA field and as a programmer.

My projects:

Android application for managing tasks and shopping list: ToDoApp

Mobile application made in Python 3.8 using the KIVY framework, dedicated to the android system devices, but after compilation it should also work on desktop devices. The app consists of a simple interface - a toolbar that allows the user to navigate between the two available windows: the list of tasks and the list of products to buy. User is able to add/remove tasks and shopping products. Link to GitHub repository: To-Do Mobile App repository
